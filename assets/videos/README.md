# Video de ejemplo para pruebas

Por favor, coloca un video de ejemplo en esta carpeta con el nombre `sample_video.mp4` para realizar pruebas de detección de pose.

El video debe contener una persona realizando alguna actividad física donde se pueda ver el cuerpo completo.

Recomendaciones:
- Formato: MP4
- Duración: 10-30 segundos
- Resolución: 720p o superior
- Persona visible de cuerpo completo
